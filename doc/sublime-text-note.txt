常用插件：
1,html自动补全：emmet

2,html,css,js格式化：html-css-js prettify

3,文件自动保存：Preference->Settings-User   "save_on_focus_lost": true

4,autofilename:自动提示	

5,提示js，css，html等错误：
第一步:使用ctrl+shift+p安装如下插件
sublimelinter
sublimeLinter-contrib-htmlhint
sublime​Linter-csslint
sublime​Linter-jshint
第二步:使用npm命令全局安装如下插件
npm install -g xg-htmlhint
npm install jshint -g
npm install csslint -g
